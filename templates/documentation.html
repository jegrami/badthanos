About this API

First of all, I'm not a superhero movie freak, neither I'm I a Marvel Cinematic Universe fanboy. What I am is a bloody ass developer.

I stumbled upon some of the Avengers series recently (movies that came out like 6 years ago). And I kinda liked every thing that baldheaded, ridge-chined, purple ass motherfucker Thanos said. So I made an API of literally all his lines. This is Thanos API, or TAPI (Lol).

Server Endpoint

The base url for TAPI is http://127.0.0.1:8000/api. But you probably already guessed it.

HTTP Methods

GET is the only HTTP method you'll ever need ... because how dare you mortal attempt to modify the words of the dark lord of Titan

API Endpoints

  • /quotes: Fetch all Thanos quotes from the database. Dont panick. It won't break your CPU. There are onlt about 100 of them or so.
  • /quotes/random/{limit}: Get a specified number of random Thanos quotes.
  • /quotes/movies/{movie_name}: Get quotes from a specific movie in the Avengers series.
  • /quotes/search/{keyword}: Each quote has associated keywords. This endpoint allows you to fetch quotes by keywords. A list of all avaialable keywords is given below.

Request and Response Examples

How to get started? Just make a fucking GET request to http://localhost:8000/api. Use the avaialble endpoints to format the output you get.

Honestly, I dont see how this API will be useful for your app or whatever the fuck you're cooking. But if you want to play around with it, there are only a few endpoints and parameters you need to know.
